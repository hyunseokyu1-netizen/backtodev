# 🎮 RPG-Style Pixel Village Blog Integration Guide

## 1. Project Overview
- **Goal:** Create an interactive 2D pixel-art RPG town experience within an existing blog as a new dedicated menu/route (e.g., `/village`), while preserving all existing post data, Markdown structure, and SEO benefits.
- **Visual Style:** 90s classic 2D top-down RPG (e.g., Pokémon, Stardew Valley aesthetic) rendered crisp and sharp without blur.
- **Technical Stack:** Next.js/React (Existing Blog Framework) + **Three.js** (using `OrthographicCamera` for a perfect 2.5D top-down perspective).

---

## 2. Core Concepts & Mapping

### 🗺️ Scene 1: The Village Square (Main Hub)
- **Starting Point:** The player spawns at the center of a lush green pixelated village map.
- **Buildings (Menus):**
  1. **My Home (Profile):** Entering this house displays user information, career logs, and skills inside a cozy pixel room.
  2. **The Library (Posts):** The central hub for all blog posts. Entering this building transitions the player to the Library interior.
  3. **The Lab / Smithy (Projects):** Displays an archive of side projects and GitHub repositories.

### 📚 Scene 2: The Library Interior (Post Archives)
- **Environment:** An indoor map filled with bookshelves.
- **Bookshelves (Categories):** Each bookshelf or area represents a specific post category (e.g., Tech, Design, Life).
- **Interaction:** Approaching a bookshelf and pressing `Spacebar` opens a clean, retro-themed HTML modal overlay displaying the list of articles in that category. Clicking an article navigates the user to the existing blog post read page.
